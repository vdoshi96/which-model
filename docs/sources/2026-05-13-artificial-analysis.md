Implementation-relevant observations:

- Reasoning/effort variants appear as separate model rows when benchmarked.
- API rows include stable IDs, provider metadata, evaluations, pricing, output speed, and latency.
- API use requires attribution and server-side caching; keys should not be exposed client-side.
