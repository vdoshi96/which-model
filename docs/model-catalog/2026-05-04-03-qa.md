## Residual Risks

- Model pricing and context windows are volatile. Each row includes `lastVerified: "2026-05-04"` and source URLs so the catalog can be refreshed intentionally.
- Some provider prices are tiered by prompt length, region, cache state, batch mode, or hosted deployment. The catalog stores a single standard display price and keeps caveats in row notes/source docs.
- Open-weight Llama rows use hosted GroqCloud pricing because Meta does not define a universal API token price.
